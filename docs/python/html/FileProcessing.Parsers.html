FileProcessing.Parsers.CoNLLU module

INFORMATIONS GENERALES

Cours de Python - M1 Sciences du Langage parcours Industries de la Langue Projet de fin de semestre

Le parser et producer “CoNLLU” permet : - de traiter un fichier CoNLL-U en entrée - de fournir une sortie d’un texte (tout ou partie) au format CoNLL-U

Le format CoNLL-U est détaillé ici : https://universaldependencies.org/format.html
