_check_arguments(type: Callable, doc, expectedArgs: set, **kwargs)[source]

Valide les arguments fournis pour créer le document.

Paramètres:

  • type (Callable) – type attendu pour le document d’entrée

  • doc (TokenizedDocument ou subclass) – document depuis lequel on construit l’objet

  • expectedArgs (set) – set contenant tous les arguments attendus pour construire l’objet

Lève:

  • TypeError – Si le type de document ne correspond à celui attendu

  • ValueError – Si tous les arguments attendus n’ont pas été fournis

  • IndexError – Lorsque les listes ne font pas la même taille (nombre de tokens)
